# cvit

Aplikacja webowa ułatwiająca przygotowanie CV dla stanowisk IT.

## Cel

Umożliwienie tworzenia wielu wersji CV dopasowanych do konkretnych stanowisk i firm, z eksportem do PDF.

## Zakres funkcjonalny

- Rejestracja i logowanie użytkowników
- Tworzenie i edycja wielu wersji CV (ogólne, pod stanowisko, pod konkretną firmę)
- Sekcje CV: dane osobowe, podsumowanie, technologie, projekty IT, doświadczenie, wykształcenie, certyfikaty, języki obce
- Oznaczanie CV jako „pod firmę X / ofertę Y" (nazwa firmy, link do ogłoszenia)
- Eksport CV do PDF

## Technologie

- **Backend:** NestJS, PostgreSQL
- **Frontend:** React (TypeScript)
- **Kontrola wersji:** Git

## Wymagania do zaliczenia

- W połowie semestru należy przekazać informację o postępach w projekcie
- Na zakończenie należy przygotować raport PDF zawierający zakres projektu, założenia, opis aplikacji, testy użytkowe, zrzuty ekranu, podsumowanie wykonanych prac oraz elementy pozostawione do dalszej realizacji
